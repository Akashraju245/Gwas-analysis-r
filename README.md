#GWAS Data Exploration using R

This repository contains a simple R script for exploring GWAS data stored in an Excel file.

## Features
- Load GWAS Excel data
- View first rows
- Basic summary statistics
- Simple p-value distribution plot

## Files
- `gwas_analysis.R` – main R script
- `data.xlsx` – GWAS dataset (Excel)
- `README.md` – project description

## How to Run
1. Open R or RStudio
2. Set working directory to this folder
3. Run:
```r
source("gwas_analysis.R")
```

## Skills Learned
- Reading Excel files in R
- Basic GWAS data inspection
- Simple visualization
